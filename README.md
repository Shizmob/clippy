clippy
======
Easy Python 2 clipboard management.
-----------------------------------


### copy to
```python
>>> clip.set(u'nice meme')    # accepts unicode or str
```

### get from
```python
>>> clip.get()                # returns unicode or None
u'nice meme'
```

### clear
```python
>>> clip.clear()
>>> clip.get()
None
```
